📋 SETUP (बस 3 steps!)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Telegram Bot Token प्राप्त करें
────────────────────────────────────────
• Telegram में @BotFather को message करें
• /newbot दबाइए
• Bot का नाम दें (जैसे: "My Riddle Game")
• Username दें (जैसे: "my_riddle_bot")
• TOKEN copy करें (बहुत महत्वपूर्ण!)

STEP 2: Code में Token डालें
────────────────────────────
• File: ultimate_telegram_game.py
• Line 571 को ढूंढें:
  TOKEN = "YOUR_TELEGRAM_BOT_TOKEN_HERE"
• अपना token डालें:
  TOKEN = "123456:ABCdef-ghIJKLmnoPQRSTUVwxyz"

STEP 3: Bot को Run करें
────────────────────────
$ pip install python-telegram-bot
$ python ultimate_telegram_game.py

✅ DONE! अब Telegram में अपना bot search करें और /start दबाइए!

🎯 HOW TO PLAY
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. /play दबाइए
2. Difficulty चुनें (Easy/Medium/Hard)
3. Riddle को पढ़िए
4. 4 में से 1 option चुनिए
   - या Hint लीजिए (2 hints available)
5. Result देखिए
   ✅ Correct → Points + Achievement check
   ❌ Wrong → सही जवाब बताया जाता है
6. अगला Riddle खेलें

❌ Bot respond नहीं कर रहा?
✓ Token सही है क्या? BotFather में /getme से check करें
✓ Internet connection ठीक है?
✓ Bot को /start से allow किया है?

❌ "ModuleNotFoundError" error?
✓ pip install --upgrade python-telegram-bot

❌ Database locked?
✓ rm ultimate_riddle_game.db (फिर से start करें)

❌ Bot अचानक बंद हुआ?
✓ Screen में run करें: screen -S game
✓ python ultimate_telegram_game.py
✓ Ctrl+A, फिर D (detach करने के लिए)
